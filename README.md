Sistem ini adalah sebuah iplementasi dalam program menggunakan bahasa pemrograman Python yang dihubungkan dengan file CSV yang berisikan data prediksi tanah longsor dari Github. Isi dari file CSV tersebut berupa data-data daerah yang ada di Sulawesi Tengah. Kemudian dalam file yang telah dihubungkan dengan file CSV tersebut memberikan beberapa output yang berasal dari program Python, diantaranya adalah informasi dataset, statistik deksriptif, analisis tingkat risiko, rata-rata risiko per kabupaten, trend risiko tahunan, analisis faktor penyebab, clustering wilayah berdasarkan karakteristik, dan bisa mengenerating hasil visualisasi yang berupa gambar yang disimpan sebagai 'analisis_longsor_sulawesi_tengah.png'. Jadi ada 2 file yang akan saling dihubungkan yaitu file dengan nama 'prediksi_longsor_sulawesi_tengah' dengan format file csv dan file dengan nama 'prediksi tanah longsor' dengan format file py.
